What is HTML?

The first web page was written in HTML (HyperText Markup Language). It is the standart markup language used to create web pages. It structures content on the web, allowing users to display text, images, and other elements

How to use HTML?

HTML uses tags to define elements. tags are enclosed in angle brackets. Elements can include headlines, paragraphs, links, images, and lists.
